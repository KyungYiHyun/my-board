# 개인 게시판 프로젝트
## 1. 프로젝트 개요
- 주제 : 넣고 싶은 기능을 구현하고 테스트해보는 게시판 개인 프로젝트
- 개발시작 : 2024-08~

## 2. 개발 환경
- Front-End : React
- Back-End : Java21, SpringBoot 3.4.8, MyBatis, MySQL
- 버전 관리 : Git
